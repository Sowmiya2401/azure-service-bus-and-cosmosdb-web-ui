# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-29

### Added
- Initial release of Azure Service Bus and Cosmos DB Management UI
- Service Bus topic and subscription management
- Dead Letter Queue (DLQ) viewer and message resubmission
- Active message inspection without consumption
- Cosmos DB database, container, and item explorer
- Custom SQL query execution for Cosmos DB
- Secure authentication with NextAuth.js v5
- Responsive design with Tailwind CSS
- Azure App Service deployment configuration
- Bicep infrastructure as code templates
- Azure DevOps pipeline configuration
- PowerShell deployment scripts
- Environment-based configuration with templates
- Comprehensive documentation

### Security
- Environment variable configuration for all secrets
- Support for Azure managed identities
- No hardcoded secrets in source code
- Secure session management
- Input validation and sanitization

### Documentation
- Complete README with setup and deployment instructions
- API documentation
- Security guidelines
- Contributing guidelines
- Code of Conduct
